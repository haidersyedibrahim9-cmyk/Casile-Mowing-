# Casile Mowing - Professional Website

A modern, responsive website for Casile Mowing featuring:
- Service showcase
- Online booking system
- Customer testimonials
- Gallery with before/after photos
- Contact forms
- Service area coverage

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Responsive Design** - Mobile-first approach
- **Booking System** - Easy service scheduling
- **Testimonials** - Build trust with customer reviews
- **Gallery** - Showcase your best work
- **Contact Integration** - Multiple ways to get in touch

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion for animations
- React Hook Form for forms
